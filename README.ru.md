# 🦆 Duckly

Duckly это игривое приложение на Flutter, которое показывает новое случайное изображение утки при каждом обновлении. Нажимайте или свайпайте, чтобы генерировать милых уток и наслаждаться весёлым звуком кряканья!

![DucklyDemo](https://github.com/KrishBharadwaj5678/Duckly/raw/main/DucklyDemo.jpeg)

## ✨ Возможности

| Возможность                       | Описание                                                             |
| --------------------------------- | -------------------------------------------------------------------- |
| 🦆 **Случайные изображения уток** | Получает милые случайные изображения уток                            |
| 🎵 **Звук кряканья**              | Проигрывает забавный звук при каждом обновлении или нажатии          |
| 🔄 **Обновление свайпом**         | Проведите вниз, чтобы мгновенно получить новую утку                  |
| 🖱️ **Кнопка генерации**          | Нажмите кнопку **Another Duck**, чтобы получить новую утку со звуком |
| ⏳ **Скелетная загрузка**          | Показывает плейсхолдер загрузки во время получения изображения       |
| 📱 **Адаптивный дизайн**          | Работает на всех размерах экранов с современным и чистым интерфейсом |

| Технология             | Назначение                                                       |
