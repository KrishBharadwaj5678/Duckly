# 🦆 Duckly

Duckly is a fun Flutter app that shows a random duck image every time you refresh. You can also play a cute quack sound while generating new ducks. Perfect for a lighthearted distraction or practicing Flutter development!

![Duckly Screenshot](https://github.com/KrishBharadwaj5678/Duckly/raw/main/DucklyDemo.jpeg)

## ✨ Features

| Feature                   | Description                                                       |
| ------------------------- | ----------------------------------------------------------------- |
| 🦆 **Random Duck Images** | Fetches cute, random duck images.                                 |
| 🎵 **Quack Sound**        | Plays a fun quack sound every time you refresh or tap the button. |
| 🔄 **Pull-to-Refresh**    | Swipe down to generate a new duck image instantly.                |
| 🖱️ **Generate Button**    | Tap the **Another Duck** button to get a new duck with sound.     |
| ⏳ **Skeleton Loading**   | Shows a loading placeholder while the image is being fetched.     |
| 📱 **Responsive Design**  | Works on all screen sizes with a modern and clean UI.             |

---

## 🛠️ Tech Stack

| Technology             | Purpose                                               |
| ---------------------- | ----------------------------------------------------- |
| 🐦 **Flutter**         | Framework for building the cross-platform mobile app. |
| 💻 **Dart**            | Programming language used with Flutter.               |
| 🎵 **audioplayers**    | Plays the quack sound effect.                         |
| 🌐 **http**            | Handles API requests to fetch random duck images.     |
| ⏳ **skeletonizer**    | Displays skeleton placeholders while loading images.  |
| 🎨 **Material Design** | Provides UI components and consistent styling.        |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/KrishBharadwaj5678/Duckly.git
```

### 2️⃣ Navigate to project directory

```bash
cd Duckly
```

### 3️⃣ Install dependencies

```bash
flutter pub get
```

### 4️⃣ Run the app

```bash
flutter run
```
