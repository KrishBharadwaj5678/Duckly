# 🦆 Duckly

Duckly は、Flutter製の楽しいアプリで、更新するたびにランダムなアヒルの画像を表示します。タップやスワイプでかわいいアヒルを生成しながら、楽しい「クワック」音も楽しめます！

![DucklyDemo](https://github.com/KrishBharadwaj5678/Duckly/raw/main/DucklyDemo.jpeg)

## ✨ 特徴

| 機能                | 説明                                 |
| ----------------- | ---------------------------------- |
| 🦆 **ランダムなアヒル画像** | かわいいランダムなアヒル画像を取得                  |
| 🎵 **クワック音**      | 更新やボタンタップ時に楽しいクワック音を再生             |
| 🔄 **プルリフレッシュ**   | 下にスワイプして新しいアヒル画像を取得                |
| 🖱️ **生成ボタン**     | **Another Duck** ボタンで音付きの新しいアヒルを生成 |
| ⏳ **スケルトンローディング** | 画像取得中にプレースホルダーを表示                  |
| 📱 **レスポンシブデザイン** | すべての画面サイズに対応したモダンUI                |
