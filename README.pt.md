# 🦆 Duckly

Duckly é um aplicativo divertido feito em Flutter que exibe uma nova imagem aleatória de pato sempre que você atualiza. Toque ou deslize para gerar patos fofos enquanto aproveita um som divertido de “quack”!

![DucklyDemo](https://github.com/KrishBharadwaj5678/Duckly/raw/main/DucklyDemo.jpeg)

## ✨ Funcionalidades

| Funcionalidade                    | Descrição                                                                 |
| --------------------------------- | ------------------------------------------------------------------------- |
| 🦆 **Imagens aleatórias de pato** | Busca imagens fofas de patos aleatórios                                   |
| 🎵 **Som de quack**               | Reproduz um som divertido de quack sempre que atualizar ou tocar no botão |
| 🔄 **Pull-to-refresh**            | Deslize para baixo para gerar uma nova imagem de pato instantaneamente    |
| 🖱️ **Botão de geração**          | Toque no botão **Another Duck** para obter um novo pato com som           |
| ⏳ **Skeleton loading**            | Exibe um placeholder de carregamento enquanto a imagem é carregada        |
| 📱 **Design responsivo**          | Funciona em todos os tamanhos de tela com uma interface moderna e limpa   |
